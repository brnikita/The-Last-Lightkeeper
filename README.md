# Последний Смотритель (The Last Lightkeeper)

Сюжетная одиночная игра от третьего лица для браузера: Эли возвращается на покинутый
остров, чтобы собрать разобранную линзу Френеля и в последний раз зажечь маяк деда.
Исследование, записки, старые радиозаписи и шторм в финале. Не хоррор —
меланхолично и тепло, как Firewatch / RiME.

## Запуск

```bash
npm install
npm run dev        # http://localhost:5173
npm run build      # production-сборка в dist/
npm run preview    # предпросмотр сборки
```

## Управление

- **WASD / стрелки** — движение, **Shift** — бег
- **Мышь** (клик по экрану — захват) — камера
- **E** — взаимодействие / закрыть записку
- **Tab** — инвентарь
- На мобильных: виртуальный джойстик + кнопка E

## Прохождение (для проверки)

1. Интро на пристани → фрагмент линзы №1 на краю мостков.
2. Деревня: ручка от радио на рыбном рынке → приёмник в доме деда (Марта объяснит
   задачу) → жестянка с ключом от маяка там же → фрагмент №2 у колодца.
3. Лес (восток): сторожка, журнал, запись деда, фрагмент №3 у пня.
4. Бухта (запад): лодка «Чайка», запись деда, фрагмент №4 в лодке.
5. Маяк на северном холме: вахтенный журнал в сундуке, дверь (нужны ключ + 4 фрагмента) →
   финал: шторм, сборка линзы, зажигание, корабль.

Сохранение автоматическое (localStorage), «Продолжить» в главном меню.

## Технологии

three.js r184 (WebGL) · Rapier 0.19 (kinematic character controller) ·
pmndrs/postprocessing (SMAA, Bloom, Vignette, ACES) · Howler (позиционный звук) · Vite
